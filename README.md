## 🌟 About The Project

StudyVault Pro is a fully browser-based study resource organizer built
using pure **HTML, CSS, and Vanilla JavaScript** — no frameworks, no
installations, no internet required after loading fonts.

Students often struggle to keep their study resources, notes, and
progress organized across multiple subjects. StudyVault Pro solves this
by bringing everything into one clean, beautiful interface that works
right in your browser — just open `index.html` and start studying!

## ✨ Features

### 📂 Resource Organizer
- Save, tag, search, filter and sort study links
- Supports 10 subjects — Math, Science, Physics, Chemistry,
  Biology, Programming, English, History, Geography & More
- Favicon preview for every resource
- Priority levels — 🔥 High, ⚡ Medium, 🌿 Low
- Per-card color themes (8 colors)
- Inline editing — edit title, URL, notes directly on the card
- Progress bar per resource with draggable slider

### 📝 Rich Notes Editor
- Full pop-up editor on every resource card
- Bold, italic, underline, strikethrough formatting
- Headings (H1–H4), font sizes, text & highlight colors
- Bullet lists, numbered lists, blockquotes, dividers
- Insert tables, code blocks, hyperlinks & images
- Upload PDF, TXT or Markdown files
- Export notes as `.html` or `.txt`
- Print / Save as PDF
- Copy to clipboard
- Auto-save every 2 seconds

### 🍅 Pomodoro Timer
- Animated circular ring timer
- Three modes — Focus, Short Break, Long Break
- Customizable durations
- Auto mode switching when timer ends
- Round counter (1–4)
- Daily session counter & streak tracker 🔥

### 📊 Subject Progress Tracker
- Visual mastery bars for every subject
- Draggable progress sliders per card and per subject
- Overall mastery percentage
- Automatic study activity log

### 🃏 Flashcard System
- Create Q&A flashcards manually per resource
- Auto-generate flashcards from your notes with one click ✨
- Smooth 3D card-flip animation
- Easy / Hard rating system
- Shuffle deck
- Live score tracking

### 🦉 Owl Mascot
- Cute animated owl with glasses, book & graduation cap
- Personalized time-based greeting dialogues on opening
- Sleepy goodbye animation on closing

### 🎨 Smart UI/UX
- Auto day/night theme based on your local time
- Manual dark/light mode toggle
- Live clock with full date display
- Animated background orbs
- Handwritten Caveat font + Nunito body font
- Auto-save with localStorage — no account needed
- Fully responsive design

## ⚡ How to Run

### Option 1 — Direct (Simplest)
1. Download or clone this repository
2. Open the `StudyVault` folder
3. Double-click `index.html`
4. It opens in your browser — done! ✅

### Option 2 — VS Code Live Server
1. Open the folder in VS Code
2. Install the **Live Server** extension
3. Right-click `index.html`
4. Click **"Open with Live Server"**
5. Opens at `http://127.0.0.1:5500` with auto-refresh ✅

### Option 3 — GitHub Pages
Visit the live link:
```
https://YOUR_USERNAME.github.io/StudyVault
```
